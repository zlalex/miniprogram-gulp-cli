# gulp

> gulp配置项

`index.js` gulp.file.js

`image.js` 处理图片, 压缩

`paste.js` 复制移动文件

`script.js` 处理js, es6编译、压缩

`style.js` 处理css, scss编译、压缩

`template.js` 处理jade, html转wxml
